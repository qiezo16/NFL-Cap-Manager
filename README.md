# NFL-Cap-Manager

An interactive salary cap simulator for NFL teams that models player contracts
and computes cap hit, dead money, and cap savings under roster moves such as
cuts and trades.

## Status
Work in progress — core cap calculation engine under development.

## Goals
- Model NFL player contracts
- Simulate cap impact of roster moves
- Support what-if scenarios

## Assumptions
- Simplified pre–June 1 cut/trade rules
- Single-season cap focus (MVP)
